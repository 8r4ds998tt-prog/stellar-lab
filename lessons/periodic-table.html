Section 01 // Core Concept

How it's organised

Elements are arranged by atomic number into a grid that groups together elements with similar properties.

Groups = columns

The vertical columns. Elements in a group have the same number of outer-shell electrons, so they react in similar ways.

Periods = rows

The horizontal rows. The period number tells you how many electron shells the atoms have.

Metals vs non-metals

Metals sit on the left and centre; non-metals on the right. A zig-zag line separates them.

Families

Group 1 = alkali metals, Group 7 = halogens, Group 0 = noble gases.

Section 03 // Encrypted: GCSE Stretch

Advanced data

🔒 Trends & why groups behave alike

Elements in a group react similarly because they have the same number of outer electrons — and it's the outer electrons that drive reactions.

Group 1 (alkali metals): reactivity increases down the group — the outer electron is lost more easily.
Group 7 (halogens): reactivity decreases down the group — it's harder to gain an electron.
Group 0 (noble gases): full outer shells → stable → almost no reactions.

History: Mendeleev arranged the table in 1869 and even left gaps for elements not yet discovered — then correctly predicted their properties.
