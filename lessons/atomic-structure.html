Section 02 // The Numbers

Atomic number & mass number

Two numbers next to a symbol tell you exactly what an atom is made of.

2311
Na

Top = mass number (23) → protons + neutrons.

Bottom = atomic number (11) → number of protons. This decides which element it is.

So sodium has 11 protons, 11 electrons and 23 − 11 = 12 neutrons.

Section 05 // Encrypted: GCSE Stretch

Advanced data

🔒 Electron shells, groups & isotopes

Electron shells fill in order. For the first 20 elements the shells hold 2, 8, 8 electrons. So sodium (11) is 2, 8, 1.

Group number = electrons in the outer shell. Sodium has 1 outer electron → Group 1.
Period number = number of shells. Sodium uses 3 shells → Period 3.

Isotopes are atoms of the same element with different numbers of neutrons — same protons, different mass. Chlorine has two: Cl-35 and Cl-37, which is why its average (relative atomic) mass is 35.5.
